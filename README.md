# 东林的诗集

## 序：

在这个诗歌已死的时代，还笨拙地以诗言意，不可不说是复古的生活方式。固然，诗歌的水平不一定高，毕竟诗歌的高峰就在盛唐，几乎把人类所有的情感写到极致，“今人不见古时月，今月曾经照古人”，我们后人写诗的意象无非是前人的重新排列组合，但是这种情感和情绪的“重新发明”依然是我们作为一个个体的独特体验，珍贵之处不比其他的体验差。
诗以言志，歌以咏怀，文以载道。宏大时代下每个平凡的普通人的声音，谁说千百年以后，不会被后人拾起来细细品味呢？


## 总结：
2019 年：18 首诗（北京中秋夜望月，阅兵，无题，追求之歌，北大夜游，你很漂亮，为何偏偏是你？，无题，孤独的诗，没有孤独二字，无题，叶落，北京初雪，无题，银河璀璨，你是人间光，无题，平安夜喜乐）

2020 年：56 首诗（相思，致相逢，元宵望月，我要你，油菜花开，春来，春分，憾错过，无题，下雨组诗，四月四，赏牡丹，Never Enough，伤离别，星月夜，小桥望月，祝新婚快乐，相思苦，仲夏夜之梦，念君归，河岸望月，睡仙歌，金鸡桥旁，曾经爱过，散步闲思，且望天，无题，月上仙，佛曰不可求佛，晚风吹，遇美人，遛狗闲情，侠客行，无题，人本孤独，等待，无题，七夕，无题，无题，秋分歌，中秋节望月，相见欢，秋天的秘密，佳人，树遇冷秋，长安街叶落如雨，许相思，双十一有感，梦在深秋，夜跑，无题，月冷，无题，遥相思，平安夜的月）

2021 年：46 首诗（生日快乐，无题，长安街夜望月，出门见雪，遇月，游颐和园记，上弦月，元宵・正月十五月，论勾践，一期一会，错过，深夜寄思，春花开，清明，夜归，Someone like you~，今安在？，深夜有相思，长歌行，遇佳人，北方的月亮，且莫悲，相思，想家，“情” 诗，情人怨，夜思，我恋爱了，然后又失恋了，莫相思，月圆人离别 -- 师兄毕业快乐～，初见，夜雨，不见不散，辛丑年六月送师兄，不如怀念，送别，一起去看流星雨，七夕，空相思，无题，且当歌，中秋月，无题）

2022 年：8 首诗（思念，去他妈的爱情，致相逢，无题，无题，中秋节操场散步独思，北医生日快乐！，无题）

2023 年：9 首诗（想你，遗憾，怨，见月，长庚凌月有感，无题，恨将别，胧月，无月夜独吟）

2024 年：20 首诗（行道难，北大春日初见，春日相思，海棠花开，恨离别，青丝，静园酣睡，将登泰山，爬泰山有感，登泰山见瀑布，泰山赏月，中元节夜宿于玉皇顶，玉皇顶看日出失败，我与明月独酌，中秋节操场跑步，我与明月独酌，国庆假期不回家，未名湖边，冬日送师兄远别，月夜思）

2025 年：16 首诗（元宵月，长相思，美人赋，想你，解相思，叹平生，人生一梦，白云苍狗，七夕，糟糕，是恋爱脑上头，乙巳蛇年中秋节望月，银杏落尽有歌，卧槽风好大，读博感怀，相聚就是缘，午夜痛歌，静待花开）

2026 年：7 首诗（丙午年初雪，情人节快乐！，海浪吟，离别家乡，清明登高怀远，一人我饮酒醉，思香芋）
